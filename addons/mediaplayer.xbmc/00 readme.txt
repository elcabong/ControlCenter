Control Center Addon Readme

Required Files:
addon.xml   -    addon name/general/version info
settings.php    -     what is displayed on the settings page for this addon
addoninfo.php   -   this is only required if the addon will display info like nowplaying or send/copy here next to each room in the selection dropdown.



should be not required
addonquicklink.php   -   adds quick links that are specific to the rooms and show up to the left of the currently selected room in the top bar.
addonlinkpages.php   -    required if addonquicklink is used.  this is where the frames are created that the links will use.
